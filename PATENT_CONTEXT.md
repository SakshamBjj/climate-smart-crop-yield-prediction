# Patent Context and Scope

This repository corresponds to a published patent application titled
**“Deep Fusion Neural Network System for Crop Yield Prediction.”**

## Scope of Disclosure
- Architectural design
- Data pipeline concepts
- Modeling strategy
- Evaluation methodology

## Excluded from Public Disclosure
- Source code
- Training scripts
- Hyperparameter configurations
- Deployment logic

## Rationale
The project is currently under patent examination. Public disclosure is intentionally
limited to ensure alignment with claimed inventions while still enabling technical review
during interviews.
